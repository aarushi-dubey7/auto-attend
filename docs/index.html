Troubleshooting

  • Open this app over HTTPS (GitHub Pages is valid) or localhost.
  • Keep Bluetooth on and place teacher iPhone beacon in discoverable mode.
  • Add your GitHub Pages URL in Supabase API CORS allowed origins.
  • Schedule match rule: student_email + room_beacon_id + start_time <= current time.
